# Restaurant Sales Analysis

## Project Overview
This project focuses on analyzing restaurant sales data to gain insights into revenue, popular items,
customer purchasing patterns, and overall business performance using Exploratory Data Analysis (EDA).

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
- Total revenue calculation
- Item-wise and category-wise sales analysis
- Quantity-based item ranking
- Date-wise sales trends
- Customer purchase insights
